## How do I use it?

This project requires Java 7+. Download the latest source at:

	https://github.com/gresrun/ghorgbrowser

Example usage:

```bash
mvn clean install && java -jar target/ghorgbrowser-1.0.0-SNAPSHOT.jar server config/development.yml
```
